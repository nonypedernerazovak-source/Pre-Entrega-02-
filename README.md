# Pre-Entrega-02-
# Sistema de Turnos Médicos

# Descripción

Este trabajo consiste en un "simulador de sistema de turnos médicos".
El programa permite al usuario gestionar un turno médico , donde puede seleccionar una opción, ingresar sus datos, elegir una especialidad y realizar acciones sobre su turno.

# Funcionalidades
El sistema permite:

- Sacar un turno médico.
- Ingresar nombre y edad del paciente.
- Seleccionar una especialidad médica:
  - Oftalmología.
  - Cardiología.
  - Dermatología.
- Asignar un horario según la especialidad elegida.
- Cancelar un turno.
- Cambiar un turno.
- Salir del sistema utilizando la opción 4 o escribiendo "ESC".
